# أبو حويل للنجف

موقع ثابت (HTML/CSS/JS) لعرض موديلات النجف والأسعار مع طلب مباشر عبر واتساب.

## التشغيل محليًا
يمكنك فتح `index.html` مباشرة في المتصفح أو تشغيل خادم بسيط:

```bash
python -m http.server 8000
```

ثم زيارة:

```
http://localhost:8000
```

## تعديل الأسعار
كل الموديلات والأسعار موجودة في ملف واحد: `models.json`.
قم بتعديل قيمة `price` لأي موديل ثم احفظ الملف فقط بدون تعديل الكود.

## النشر على GitHub Pages
1. ارفع الملفات إلى الفرع `main`.
2. من إعدادات GitHub Pages اختر الفرع `main` والجذر `/`.
3. سيتم نشر الموقع تلقائيًا.
